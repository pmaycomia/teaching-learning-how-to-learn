Focused Mode

  • In focused mode, our brain is concentrating on a familiar concept or idea.
  • We use it to learn, practice, or strengthen a brain pattern.
  • It is necessary for deep work, and helps greatly in forming a new chunk.

Diffuse Mode

  • In diffuse mode, our brain is in relaxed state. We see the big picture and it is the mode where we are highly creative.
  • The brain can access broad ranging ideas to understand a new concept using an existing one stored in our long term memory .

in summary...

We use FOCUSED mode when we have to study for exams, and DIFFUSE mode when we have to understand a new concept.

In order to be an effective learner, we have to use both modes alternately.

Friends, I'd like to introduce you to your mortal enemy, procrastination.

Procrastination

My own definition of procrastination is running away from things that needs to be currently done, thinking "I can do that later".

But not actually doing it, or sometimes never.

I know a lot of us can relate and struggle with procrastination daily.

"So what is the solution?"

Pomodoro Technique

A time management method developed by Francesco Cirillo using a kitchen-timer.

How to use it:

  1. Choose a task
  2. Remove all distractions
  3. Set a timer of 25mins(default)
  4. Reward yourself with a break for 5-10mins
  5. Repeat
